📄 CV Analyzer & AI Feedback Tool

🚀 Overview

This project is a CV (Resume) Analysis Tool built with Python and Streamlit.
It allows users to upload their CV in PDF format and receive structured feedback about missing sections and improvements.

There are two versions of the application:
	•	Rule-based analyzer (cv.py)
	•	AI-powered analyzer using Google Generative AI (cv_ai.py)

⸻

✨ Features

📊 CV Analysis
	•	Extracts text from PDF using pdfplumber
	•	Detects CV language (English / Turkish)
	•	Identifies key sections:
	•	Experience
	•	Education
	•	Skills
	•	Projects

🧠 Feedback System
	•	Rule-based suggestions (basic version)
	•	AI-generated suggestions using Google Gemini (advanced version)

🌐 User Interface
	•	Built with Streamlit
	•	Simple and interactive dashboard
	•	Upload CV and instantly see results

⸻

🧰 Technologies Used
	•	Python
	•	Streamlit
	•	pdfplumber
	•	regex (re)
	•	spaCy
	•	pandas
	•	Google Generative AI (Gemini)
